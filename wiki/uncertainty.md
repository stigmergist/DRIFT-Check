# Uncertainty

Uncertainty is when the same action produces different outcomes.

This is not the same as complexity. Complexity is many ways to reach the same outcome. Uncertainty is unstable cause and effect. The distinction matters because the interventions differ.

In uncertainty, direct optimisation is usually premature. The system needs probing: bounded experiments, tighter observation, and clearer signal discrimination.

Uncertainty is not inherently bad. It is expected in exploration, innovation, and rapidly shifting environments. The risk is pretending uncertainty is already resolved and scaling decisions that were never properly tested.
