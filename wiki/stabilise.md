# Stabilise

Stabilise means restoring control and reliability when outcomes are inconsistent.

Stabilisation work is about making the system predictable enough that it can be trusted. It addresses recurring incidents, variable outcomes, reactive firefighting, and dependence on specific individuals to keep things running.

The aim is not perfection. The aim is reduced recurrence and improved control. A system is stabilising when the same issues happen less often, are easier to diagnose, and produce less collateral disruption.

**[Stabilise](stabilise.md)** comes before **[rationalise](rationalise.md)** and **[optimise](optimise.md)**. If that sequence is ignored, later interventions are built on unstable ground.
