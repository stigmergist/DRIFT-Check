# Anti-Patterns

## Common Failure Modes

### Optimising instability
- Automating broken processes
- Outcome: Faster failure

### Rationalising without alignment
- Multiple standards emerge
- Outcome: Increased inconsistency

### Acting in Probe as if Proceed
- Premature scaling
- Outcome: Expensive mistakes

### Investing in Stop contexts
- Improving low-value work
- Outcome: Waste

## Detection Heuristics

- “This should be working” → likely Stabilise issue
- “Everyone does it differently” → Rationalise issue
- “We’re not sure what’s happening” → Probe context
- “Why are we doing this?” → Stop context
