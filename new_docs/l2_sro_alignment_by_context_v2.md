# Layer 2 SRO Alignment by Context (v2)

Use this after identifying Layer 1 context and Layer 2 state.

Interpretation key:

- ✅ `Right fit`: action matches current need.
- 🔵 `Limited value`: action may help partially but misses primary constraint.
- 🟠 `Misapplied`: action conflicts with current need and likely creates drag.
- 🔴 `High risk`: action amplifies underlying weakness.
- 🚨 `Very high risk`: action actively degrades system state.
- ☢️ `Extreme risk`: action accelerates failure or locks in damage.

## Practical use

1. Confirm context first (Layer 1).
2. Identify actual state (SRO checklist).
3. Prefer actions in the Right fit cell.
4. If evidence is mixed, use bounded hybrid steps and short feedback loops.
5. If risk cells dominate, reduce scope and re-sequence before scaling.
