# DRIFT Companion Docs (v2)

This directory contains revised, clearer versions of the source docs from [docs](../docs).

## Files

- `l1_context_matrix_v2.md`  
  Layer 1 context states with observable signals, safe actions, and premature-action risks.

- `l1_context_checklist_v2.md`  
  Tick-box checklist to identify the dominant Layer 1 context: Proceed, Align, Probe, or Stop.

- `l2_sro_checklist_v2.md`  
  Tick-box checklist to identify the dominant Layer 2 action need: Stabilise, Rationalise, or Optimise.

- `l2_sro_alignment_by_context_v2.md`  
  Context-specific action-fit matrix showing what happens when action type and actual state do or do not match.

- `l2_sro_alignment_matrix_v2.md`  
  One-page baseline alignment matrix for fast action-state fit checks.

## How to use these docs together

1. Start with `l1_context_checklist_v2.md` to determine context readiness.
2. Use `l2_sro_checklist_v2.md` to identify the dominant capability state.
3. Validate fit with `l2_sro_alignment_by_context_v2.md` (or quick-check via `l2_sro_alignment_matrix_v2.md`).
4. If scores are close or contradictory, treat as mixed and run bounded probing before scaling.

## Scoring note

Checklist scores are directional, not diagnostic certainty. Observable behaviour always outweighs numerical totals.

## Icon legend

| Icon | Meaning |
|---|---|
| 🟢 | Proceed — context is clear and ready |
| 🟡 | Align — shared understanding is lacking |
| 🔍 | Probe — cause-effect is unclear |
| ⛔ | Stop — value is unclear |
| 🔧 | Stabilise — restore control and reliability |
| 📐 | Rationalise — reduce complexity |
| 🚀 | Optimise — improve performance |
| ✅ | Right fit |
| 🔵 | Limited value |
| 🟠 | Misapplied |
| 🔴 | High risk |
| 🚨 | Very high risk |
| ☢️ | Extreme risk |

## Facilitation mode (DRIFT-consistent)

When using these sheets in conversation, keep outputs minimal:

1. Name one dominant observable signal.
2. Name one key interaction or misfit.
3. State one likely consequence if unchanged.
4. Ask one forward question.

Avoid multi-label diagnosis unless explicitly requested.
